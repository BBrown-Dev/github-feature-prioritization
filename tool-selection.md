# Selection of Developer-Focused Tool: GitHub

## Overview
GitHub is a cloud-based platform used by developers to host, manage, and collaborate on code repositories. It supports version control using Git and provides tools for automation, security, and team collaboration.

## Primary Purpose
GitHub enables developers to:
- Manage source code
- Collaborate through pull requests and code reviews
- Automate CI/CD pipelines
- Track issues and project tasks
- Integrate with cloud-native tools and platforms

## Key Features
- Git-based version control
- Pull requests and code review workflows
- GitHub Actions for CI/CD automation
- Issue tracking and project boards
- Security scanning (Dependabot & CodeQL)
- Integrations with AWS, Azure, GCP, Jira, Slack, and more
